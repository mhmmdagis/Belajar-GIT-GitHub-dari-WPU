version control system (VCS) : system yang menyimpan dan mengelola kekaman perubahan dari source code
GIT: salah satu softwear yag bisa melakukan VCS
GitHub: website/layanan untuj mengelola project git

Istilah git 
- repo folder project kita
- commit rekaman/snapshot dari repo kita
- hash penanda unik pada sebuah commit
- checkout berpindah ke sebuah commit
- branch cabang bebas dari sebuah commit
- merge menggabungkan branch 
- remote sumber yang memiliki repo
- clone menggambil repo dari remote
- push mengirimkan commit ke repo
- pull menggambil commit dari repo
